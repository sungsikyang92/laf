# Lost and Found 웹 프로젝트
분실물에 대하여 습득자와 분실자를 연결해주는 웹 서비스입니다.   
분실자는 자기가 잃어 버린 물건, 습득자는 자신이 주은 분실물에 대해서 글을 올려 상대방과의 연결을 통해 회수할 수 있게 합니다.   
이를 통해 좀더 밝은 사회를 만들고자 프로젝트를 진행하였습니다.
